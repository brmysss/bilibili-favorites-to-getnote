自动发现收藏夹新增内容，并交给得到大脑解析和归档。

同步范围

默认首次检查只建立基线，不上传历史收藏。开启“同步全部日期”后，会上传所有尚未处理的现有收藏。B站轻量接口不提供收藏时间，因此无法精确按历史日期切分。

数据备份

同步基线、处理记录和日志保存在本机的 chrome.storage.local。导出的 JSON 默认不包含 API Key 和 Client ID。
